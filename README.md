# Student Scoreboard

This is a React and Vite project made for a Web Development II assignment.

## Features

- Add a new student
- View student names and marks
- Update student marks
- Show pass or fail status
- Display total students, passed students, and average marks

## Technologies Used

- React
- Vite
- JavaScript
- CSS

## How to Run

1. Open the project folder in terminal
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the local link shown in the terminal
